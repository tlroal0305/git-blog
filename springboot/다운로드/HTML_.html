단축키

  • 주석 달기 : Ctrl + /
  • 코드 이동 : Alt + ↑↓(상하 방향키)
  • 코드 복사 : Alt + Shift + ↑↓(상하 방향키)
  • 저장하기 : Ctrl + S
  • 환경설정 : Ctrl + , (쉼표)
  • 메뉴 검색하기 : Ctrl + Shift + P
  • 라이브서버로 html 파일 열기 : Alt + L, ALT + O

'HTML'이란?

HTML파일은 마크업(Markup) 언어로 된 문서로 XML의 하위 집합으로서 XML의 규칙을 따름

  • head 태그 : 문서 제목 title과 문서 정보 meta를 담고 있음
  • body 태그 : 실제 콘텐츠를 담고 있음

*'XML'이란? 태그 기반의 정보는 담는 문서 형식

기본 태그

글자 태그

글자 태그에는 '제목 태그' : H1 ~ H6 과 '본문 태그' : P(본문 문단 생성), BR(줄 바꿈), HR(수평 줄 삽입)이 있습니다.


글자 속성 태그

  1. 'b' : bold, 굵은 텍스트 (예) 예시 텍스트
  2. 'i' : italic, 기울어진 텍스트 (예) 예시 텍스트
  3. 'small' : 텍스트가 작은 효과 (예) 예시 텍스트
  4. 'sub' : 텍스트가 아래에 배치됨 (예) 예시 텍스트
  5. 'sup' : 텍스트가 위에 배치됨 (예) 예시 텍스트
  6. 'ins' : 텍스트에 밑줄 효과 (예) 예시 텍스트
  7. 'del' : 삭제 또는 가운데 줄 효과 (예) 예시 텍스트
  • 더미 문구 : 의미 없는 문구 삽입 'LOREM IPSUM'

목록(리스트) 태그

  1. OL : Order list, 순서 있는 목록 (예) 1 2 3
  2. UL : Unorder list, 순서 없는 목록 (예) ● ● ●
  3. UL : List, 항목 1개를 의미

바로가기 태그

다른 말로는 '앵커 태그', '하이퍼링크 태그'라고 합니다. 'a'

  • 페이지 안에서의 이동 : 바로가기 링크 넣는 위치에 #[페이지 이름] + 페이지 아이디
